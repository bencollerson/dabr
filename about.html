What is dabr?

  • A mobile web interface to Twitter's API
  • An open source project by @davidcarrington with inspirations from @whatleydude and awesome contributions from Terence Eden
  • Secure, storing your Twitter login details in an encrypted cookie on your machine, and never stored on the website

If you have any comments, suggestions or questions then feel free to get in touch.

About dabr.appb.in

This clone of dabr is maitained by @vsr. Source available at github.com/vsr/dabr.

Features added:

  • Displays twitter api ratelimits.
  • Option in settings to auto-reload timeline every specified number of seconds.
  • Added themes `Mainframe`, `Pale Dale`.
